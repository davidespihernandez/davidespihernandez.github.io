Scripting

Use a Groovy (java) based language to create your own programs that process your information.

Atkloud provides an API that allows programmers to manipulate databases (structure and information), non sql collections, files, execute workflows, operate with processes, notify users, etc. Any aspect of the system can be accessed or manipulated via Atkloud's API

You can also schedule scripts, and use scripts in Data Spreadsheets as macros or events (before loading record, after save...)
