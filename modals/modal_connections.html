External Connections

Atkloud allows integration with external sources

Atkloud allows linking a number of external data sources such as Relational Databases (Mysql, Postgres, Oracle, SQL Server), NonSQL databases (MongoDB), external storage drives (Amazon S3, Dropbox, Google Drive, sFTP, Windows Azure). Out of the box Atkloud provides a local MongoDB and a local internal drive.

Atkloud also allows to link to an external logging storage system or create custom logs and appenders to be used accross the system. Out of the box Atkloud provides a system log that uses Elastic Search
